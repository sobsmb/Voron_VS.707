
# Voron VS.707

Cleaned and reorganized Enderwire / Voron-style Klipper config.

## Layout
- `hardware/` board, toolhead, LEDs
- `tuning/` input shaper and accelerometers
- `features/` bed mesh, sensors, helper includes, backup macro
- `macros/` grouped by purpose: `calibration`, `filament`, `print`, `system`
- `services/` Moonraker, Crowsnest, KlipperScreen, Sonar, autocommit
- `archive/` old or disabled config
- `artifacts/` generated files kept out of Git by default
